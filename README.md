# NavVoice – AI Voice Navigation Assistant

## Problem
Visually impaired people often face difficulty while navigating because most navigation applications depend on visual maps and screens. This makes it hard for them to understand their surroundings or know their exact location while walking.

## Solution
NavVoice is a voice-based navigation assistant designed to help visually impaired individuals travel more safely and independently. The system provides real-time voice guidance and continuously informs the user about their current location and nearby landmarks.

For example:
"You have reached Bengaluru Bakery. Continue straight for 50 meters. A medical shop is on your right."

## Features
- Voice-based navigation guidance
- Continuous location updates
- Information about nearby surroundings
- Accessibility support for visually impaired users
- Simple prototype using voice output

## How It Works
1. The user starts navigation using voice commands.
2. The system processes the destination.
3. Navigation directions are generated.
4. The system announces directions and nearby landmarks using voice.

## Technologies Used
- HTML
- JavaScript
- Web Speech API
- Text-to-Speech

## Future Improvements
- Integration with real-time map services
- Obstacle detection
- Personalized voice assistant
- Offline navigation support

## Impact
NavVoice can improve mobility and independence for visually impaired individuals by providing safe and accessible navigation through voice guidance.
